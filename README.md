Getting Started
===============

Install

	$ cd ~
	$ git clone https://github.com/ilcic/config.git
	$ cd config
	$ ./install.sh -a
	$ ./install.sh -i
	$ ./install.sh -v
	$ ./install.sh -s
	$ source /root/.bashrc

For vim install please check out `vim`
